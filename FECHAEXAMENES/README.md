# Generador de Fechas de Exámenes

Este repositorio contiene el **sistema Generador de Calendarios de Exámenes**, desarrollado para automatizar la planificación de exámenes universitarios, optimizando tiempos y evitando conflictos de horarios.

Se incluyen los siguientes componentes principales:

---

## Contenido del Repositorio

- [Modelo del Dominio](/modelos/README.md)  
  Contiene los diagramas de clases, objetos y estados del sistema, junto con la descripción de la idea principal y los objetivos del generador de exámenes.

- [Casos de Uso](/casos-de-uso/README.md)  
  Incluye los casos de uso de cada actor del sistema (Alumno, Profesor, Usuario, Generador) con sus relaciones, diagramas resumidos y enlaces a las imágenes de cada caso de uso.

- [Interacciones con el Catedrático y Cliente](/interaccionesClienteYCatedratico)  
  Incluye las conversaciones, correcciones y recomendaciones recibidas tanto del cliente como del catedrático, incluyendo reuniones y seguimiento de avances.

- [Diagrama de contexto](/casos-de-uso/DiagramaDeContexto/)
  Tiene una línea la cual es posible seguir al momento de la creación del proyecto

---

## Objetivo

Automatizar la generación de calendarios de exámenes, asegurando:

- Reducción de tiempo y esfuerzo manual.
- Prevención de conflictos de horario.
- Publicación de calendarios fiables para alumnos y docentes.
- Integración con la base de datos universitaria para mantener información actualizada.

---

## Tecnologías y Herramientas

- PlantUML para diagramas.
- Markdown para documentación.
- Git y GitHub para control de versiones y colaboración.
- Conexión a base de datos universitaria (simulada en los modelos).
- Inteligencia artificial (ChatGPT, Deepseek, Claude)
